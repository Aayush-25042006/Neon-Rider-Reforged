# Neon-Rider-Reforged
Neon Rider: Reforged is an endless 2D physics-based motorcycle racing game packed into a single HTML file. Dodge obstacles, manage your speed, and spend points on energy shields as you survive a procedurally generated neon highway. Built entirely from scratch without a game engine!


# Neon Rider: Reforged 🏍️✨

A fast-paced, 2D endless motorcycle racing game built entirely in a single HTML file. Heavily inspired by classic arcade physics games like *Hill Climb Racing*, but drenched in a retro, neon-soaked synthwave aesthetic.

## 🌟 Features

*   **Zero Dependencies:** 100% Vanilla JavaScript, HTML5 Canvas, and CSS. No game engines or external libraries required.
*   **Procedural Terrain:** The neon highway generates infinitely, with hills and valleys that become more aggressive as you progress.
*   **Dynamic Speed Scaling:** The game automatically speeds up every 150 points, increasing the challenge and testing your reaction time.
*   **Physics-Based Movement:** Control your bike's pitch and momentum to stick the landings and avoid flipping.
*   **Procedural Audio:** Engine hums and 8-bit crash sound effects are generated completely dynamically using the Web Audio API—no audio files needed!
*   **Risk/Reward Mechanics:** Spend your hard-earned points to deploy an energy shield and smash through obstacles, or save them to push your high score.

## 🎮 How to Play

Survive as long as possible without crashing into an obstacle, flipping your bike, or falling through the map.

### Controls:
*   **[ ↑ ] / [ ↓ ]** : Tilt Bike Forward / Backward
*   **[ ← ] / [ → ]** : Brake / Accelerate
*   **[ SPACE ]** : Jump (Must be grounded)
*   **[ SHIFT ]** : Activate Shield (Costs 50 Points)

## 🚀 How to Run

Because this game is built into a single file, installation is instant:

1. Clone or download this repository.
2. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Click **START ENGINE** and enjoy the ride!

## 🛠️ Technologies Used

*   **HTML5 Canvas API:** For all rendering, drawing, and visual effects (glows, shadows).
*   **Vanilla JavaScript:** Custom physics, collision detection, and game loop handling.
*   **CSS3:** UI styling, gradients, and subtle CRT scanline overlays.
*   **Web Audio API:** Real-time synthesis for engine and crash sound effects.
